**Smarter Contacts Project#**
        This is a spring Boot Contact Manager Application.
  # Features      
     - User Registration $Login
     - Add Contacts
     - Update Contacts
     - Delete Contacts
     - Secure Authentication

   # Technology Used
     - java
     -Spring Boot
     -Thymeleaf
     -BootStrap
     -Jpa
     -Secruity

   # How to Run
     1. Clone the repository
     2. Open in spring Tool Suite(STS)/Eclipse
     3. Run as Spring Boot application 
     4. Open in browser: http://localhost:8081
